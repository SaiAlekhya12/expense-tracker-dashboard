# Expense Tracker

A full-stack Expense Tracker application built using Spring Boot, Spring Data JPA, H2 Database, HTML, CSS, and JavaScript.

## Features

* Add Expenses
* View Expenses
* Update Expenses
* Delete Expenses
* REST API Integration
* H2 Database Support
* Frontend User Interface

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA
* H2 Database
* Maven

### Frontend

* HTML
* CSS
* JavaScript

### Version Control

* Git
* GitHub

## API Endpoints

GET /api/expenses

POST /api/expenses

PUT /api/expenses/{id}

DELETE /api/expenses/{id}
