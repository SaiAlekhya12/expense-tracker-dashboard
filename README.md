# Expense Tracker

## Project Overview

Expense Tracker is a full-stack web application that helps users manage their daily expenses.

Users can:

* Add expenses
* View expenses
* Update expenses
* Delete expenses

The application is built using Spring Boot, Spring Data JPA, H2 Database, HTML, CSS, and JavaScript.

---

## Features

* Create Expense
* Read Expense
* Update Expense
* Delete Expense
* REST APIs
* H2 Database Integration
* Frontend User Interface

## API Endpoints

### Get All Expenses

GET /api/expenses

### Add Expense

POST /api/expenses

### Update Expense

PUT /api/expenses/{id}

### Delete Expense

DELETE /api/expenses/{id}
